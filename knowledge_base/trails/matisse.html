Java GUI Application Learning Trail

What is a GUI application?

A graphical user interface (GUI) is an application that uses the standard Java components GUI component set, Swing, and is deployed to the desktop. Like all Java code, Java GUI applications simplify the task of creating desktop applications because they can be run on any platform.

The GUI Builder in NetBeans IDE has long provided advanced GUI application support. In NetBeans IDE 5.0, this support was greatly enhanced with new tools (code named project Matisse) that makes the layout design of visual forms much easier. In NetBeans IDE 6.0, support for the Swing Application Framework and Beans Binding technologies has been added.
